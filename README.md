### Face Capture Flow

This is a demo flow for capturing or uploading facial photos.

---

### Features

- **Intro flow** (for camera permission prompt)
- **Camera capture flow**  
- **Manual upload flow**  
- **Uploading screen**  
---

### Tech Stack

- Next.js  
- TypeScript  
- Tailwind CSS  
- Lucide Icons
- MediaPipe FaceMesh (face detection)
